# SmartStudent - Student Management System

## Technologies Used

- Java
- JDBC
- MySQL
- VS Code

## Features

- Admin Login
- Add Student
- View Students
- Update Student
- Delete Student
- Search by Roll Number
- Search by Department
- Search by Marks

## Database

Database Name:

smartstudent

Table:

students

## How to Run

1. Import the SQL file into MySQL.
2. Update database credentials in `DatabaseConnection.java`.
3. Add the MySQL JDBC Connector JAR to the project.
4. Compile and run `Main.java`.
